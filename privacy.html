Privacy Policy for Image SEO Analyzer

Last Updated: [Date of submission]

1. Introduction

Welcome to Image SEO Analyzer ("the Extension"), a browser extension developed by Lutfor Pro. Your privacy is critically important to us. This privacy policy explains what data the Extension interacts with and, more importantly, what it does not do.

2. Data Collection and Usage: No Data is Collected

The Image SEO Analyzer extension is designed with a "privacy-first" approach. It does not collect, store, share, or transmit any personal or sensitive user data.

  • Local Analysis: All analysis of images (ALT text, file names, dimensions, etc.) is performed locally within your browser on the current web page you are viewing.
  • No External Communication: The extension does not send any of the analyzed page data or your browsing activity to any external servers, including those of the developer. All processing remains on your computer.
  • Local Storage for Settings: The extension uses Chrome's local storage API (`chrome.storage.local`) for one purpose only: to save your preference for the "Show on-page highlights" toggle (on or off). This setting is stored exclusively in your browser and is not accessible to anyone else.
  • Data Export Feature: The "Download CSV" feature generates a file directly in your browser from the data on the current webpage. This data is saved directly to your computer's downloads folder without being sent over the network to the developer or any third party.

3. Permissions Justification

The extension requests the minimum permissions necessary for its core functionality:

  • `activeTab` & `scripting`: Required to read the content of the web page you are currently viewing to analyze the images. This permission is only active for the tab you are on when you use the extension.
  • `storage`: Required to save the state of the on/off toggle switch between browser sessions.

4. Changes to This Policy

We may update this privacy policy in the future to reflect changes in the extension's functionality. We will notify you of any significant changes by updating the extension listing on the Chrome Web Store.

5. Contact Us

If you have any questions about this privacy policy or the practices of this extension, please contact the developer, Lutfor Pro, through the support options provided on the Chrome Web Store listing.
